Current Session
Your admin session is secured with a server-side token that expires after 8 hours. Tokens are stored in the database and verified on every request.
Security Notes
  • ✅ Admin sessions expire after 8 hours
  • ✅ All write routes require a valid admin token
  • ✅ Token verified server-side on every request
  • ✅ Role checked in DB to prevent privilege escalation
  • ✅ Passwords stored as bcrypt hashes (cost 10)
  • ⚠️ Change the default password immediately after setup
  • ⚠️ Never share your admin token
